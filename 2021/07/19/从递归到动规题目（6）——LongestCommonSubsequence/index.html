从递归到动规题目（6）

LongestCommonSubsequence

题目描述

给定两个字符串str1和str2，返回这两个字符串的最长公共子序列长度。如： str1 = "a12b3c456d",str2 = "1ef23ghi4j56k"最长公共子序列是"123456"，所以返回长度6。

算法分析

暴力递归

  • 这道题尝试模型很新颖，但是其实也不算很难，只要分清楚情况就好了。这种分类方式也是一种套路，属于两个样本的对应模型。

我们定义一个递归过程，int process(char[] str1, char[] str2, int i, int j)。这个递归过程返回：str1[0...i]范围上的字符串和str2[0...j]范围上的字符串的最长公公子序列的长度。

  • 我们来看一下递归的一般可能性分类:

    1. 最长公共子序列，一定不以str1[i]字符结尾、也一定不以str2[j]字符结尾

    2. 最长公共子序列，可能以str1[i]字符结尾、但是一定不以str2[j]字符结尾

    3. 最长公共子序列，一定不以str1[i]字符结尾、但是可能以str2[j]字符结尾

    4. 最长公共子序列，必须以str1[i]字符结尾、也必须以str2[j]字符结尾

    注意：1.2.3.4并不是完全互斥的，他们可能会有重叠的情况，很显然，2.3包含了1。
    但是可以肯定，答案不会超过这四种可能性的范围
