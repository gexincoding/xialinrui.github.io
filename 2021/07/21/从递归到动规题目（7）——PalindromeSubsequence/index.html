从递归到动规题目（7）

PalindromeSubsequence

题目描述

给定一个字符串str，返回这个字符串的最长回文子序列长度。

比如 ： str = “a12b3c43def2ghi1kpm”

最长回文子序列是“1234321”或者“123c321”，返回长度7

算法分析

法1:利用最长公公子序列思想

  • 具体流程是，先把给的字符串s1倒序成s2，答案是s1与s2的最长公公子序列的长度。

  • 代码：

  • 大概解释一下：s1与s2最长公公子序列一定是对称的，那么总体最长意味着一半也最长，一半最长且公共说明一定回文（因为两个字符穿互为倒叙关系）

  • 其中，求两个字符串的最长公公子序列的方法不再细谈。

法2:就题目本身的暴力递归

如果想不到上述的方法，我们可以思考，如何根据题目本身的含义去推出一个递归。

显然，这是一种范围上的尝试模型。

我们定义一个方法：private static int process(char[] s, int L, int R) ，这个递归表示：在s的[L…R]范围上，返回最长回文子序列的长度。

  • 注意：范围上的尝试不同情况的讨论方式是：包不包含L或者R
